# Reybex Warehouse Dashboard

Bu proje, depoların doluluk durumlarını ve içindeki ürünleri göstermek için
geliştirilmiş basit bir React uygulamasıdır.

Depolar kartlar halinde listelenir. Doluluk oranına göre renklenir.
Bir depoya tıklandığında, o depoya ait ürünler ve kapasite bilgileri görüntülenir.

## Özellikler

- Depo listeleme (kart yapısı)
- Doluluk oranına göre renkli gösterim
- Depoya tıklayınca ürün listesi
- Basit ve anlaşılır arayüz

## Kullanılan Teknolojiler ve Kütüphaneler

- React
- React DOM
- Vite
- JavaScript
- JSON (mock veri)

## Çalıştırma

Projeyi çalıştırmak için:

```bash
npm install
npm run dev
```
